# Roadmap Status - 進捗状況表示

`.docs/roadmap.md` と `.docs/tasks/*.md` を読み込み、進捗状況をサマリー表示する。

## 手順

1. `.docs/roadmap.md` を読み込む
2. `.docs/tasks/*.md` を一覧取得
3. 以下の情報を集計・表示:

## 出力フォーマット

```markdown
## ロードマップ進捗状況

### サマリー
| 状態 | 件数 |
|------|------|
| ✅ 完了 | X |
| 🔄 進行中 | X |
| ⬜ 未着手 | X |
| **合計** | **X** |

### マイルストーン別
| マイルストーン | 完了 | 進行中 | 未着手 | 進捗率 |
|---------------|------|--------|--------|--------|
| M1: MVP | X | X | X | XX% |
| M2: コア機能拡張 | X | X | X | XX% |
| M3: 追加機能 | X | X | X | XX% |
| M4: 仕上げ | X | X | X | XX% |

### 次のタスク候補
依存関係が解消されており、着手可能なタスク:

1. **T-M{X}-{N}**: {タスク名}
   - マイルストーン: M{X}
   - 依存: なし（または依存タスク完了済み）

2. **T-M{X}-{N}**: {タスク名}
   - マイルストーン: M{X}
   - 依存: T-M{X}-{N}（完了済み）
```

## 注意事項

- roadmap.md が存在しない場合はエラーメッセージを表示
- タスクファイルと roadmap.md の状態に不整合がある場合は警告を表示
