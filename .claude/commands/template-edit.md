# Template Edit - テンプレート編集モード

このワークフローテンプレート自体を編集・改善するためのモードです。

## 概要

このコマンドを実行すると、通常の「コマンド待機ルール」を一時的に解除し、テンプレートの編集作業を自由に行えるようになります。

## 編集対象

テンプレートを構成する以下のファイル・ディレクトリが編集対象です：

- `CLAUDE.md` - プロジェクト指示ファイル
- `.claude/WORKFLOW.md` - ワークフロー詳細仕様
- `.claude/commands/` - スキルコマンド群
- `.claude/skills/` - スキル群

## 実行手順

### 1. 現在のテンプレート構成を確認

```bash
# テンプレート関連ファイルの一覧
ls -la CLAUDE.md
ls -la .claude/
ls -la .claude/commands/
ls -la .claude/skills/
```

### 2. ユーザーに編集内容を確認

以下のように確認してください：

```
## テンプレート編集モード

編集対象のファイル一覧を表示しました。
どのような編集・改善を行いますか？

例:
- 新しいコマンドの追加
- 既存コマンドの修正
- ワークフローの改善
- CLAUDE.md の更新
```

### 3. 作業実行

ユーザーの指示に従い、自由に編集作業を行ってください。

**このモードでは通常の「コマンド待機ルール」は適用されません。**

### 4. 作業完了後

編集内容をまとめて報告してください：

```
## テンプレート編集完了

### 変更内容
- [変更したファイル]: [変更内容の概要]

### 次のステップ
変更をコミットする場合は指示してください。
```
