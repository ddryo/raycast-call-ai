# T-M1-4: LocalStorage 永続化層の実装

| 依存 | T-M1-1 |

## 説明
Raycast LocalStorage を使って会話データの保存・復元を行うモジュールを実装する。Phase 2 のスレッド管理に備え、キー設計を `ask-ai:messages:{threadId}` 形式とする。

## 作業内容
1. `src/storage/conversation.ts` を作成
2. `saveMessages()` 関数を実装:
   - 引数: threadId, Message[]
   - LocalStorage キー: `ask-ai:messages:{threadId}`
   - JSON.stringify して保存
3. `loadMessages()` 関数を実装:
   - 引数: threadId
   - LocalStorage から取得し JSON.parse して返す
   - キーが存在しない場合は空配列を返す
4. `clearMessages()` 関数を実装:
   - 引数: threadId
   - 該当キーを removeItem で削除
5. Phase 1 ではデフォルト threadId として `"default"` を使用する定数を定義

## タスクのチェックリスト
- [ ] saveMessages() が Message[] を JSON 化して LocalStorage に保存する
- [ ] loadMessages() が LocalStorage から復元し Message[] を返す
- [ ] clearMessages() が指定 threadId の会話データを削除する
- [ ] キーが存在しない場合に空配列が返る
- [ ] デフォルト threadId 定数が定義されている
