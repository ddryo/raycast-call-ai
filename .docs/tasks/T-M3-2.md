# T-M3-2: 二重送信防止の実装

| 項目 | 内容 |
|------|------|
| マイルストーン | M3 |
| 依存 | T-M2-2 |
| 要件ID | FR-007 |

## 説明
API 応答待ち中にユーザーが再度メッセージを送信することを防止する。isLoading 状態を活用して入力ロックとローディング表示を実装する。

## 作業内容
1. sendMessage() 冒頭で isLoading チェックを追加:
   - isLoading が true の場合は早期リターン
2. List の isLoading プロパティに状態をバインド（SearchBar 下にローディングバー表示）
3. isLoading が true の間:
   - SearchBar での送信アクションを無効化
   - ActionPanel の送信関連アクションを無効化またはグレーアウト

## タスクのチェックリスト
- [ ] API 応答待ち中に再送信しても無視される
- [ ] ローディングバーが SearchBar の下に表示される
- [ ] 応答受信後に入力ロックが解除される
- [ ] エラー発生後も入力ロックが解除される
