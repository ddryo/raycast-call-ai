# T-M3-4: メッセージコピー機能の実装

| 項目 | 内容 |
|------|------|
| マイルストーン | M3 |
| 依存 | T-M2-2 |
| 要件ID | FR-009 |

## 説明
選択中のメッセージの内容をクリップボードにコピーする機能を実装する。

## 作業内容
1. 各 List.Item の ActionPanel に「内容をコピー」アクションを追加
2. Action.CopyToClipboard を使用:
   - content に選択メッセージの content を渡す
3. キーボードショートカットを設定（Cmd+C など、Raycast の慣習に従う）

## タスクのチェックリスト
- [ ] ActionPanel に「内容をコピー」アクションが表示される
- [ ] 選択メッセージの内容がクリップボードにコピーされる
- [ ] コピー成功時に HUD またはToast が表示される
