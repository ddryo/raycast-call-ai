# T-M3-3: 会話クリア機能の実装

| 項目 | 内容 |
|------|------|
| マイルストーン | M3 |
| 依存 | T-M2-3 |
| 要件ID | FR-008 |

## 説明
現在の会話履歴を全削除する機能を実装する。ActionPanel からアクションを選択し、確認ダイアログの後に会話をクリアする。

## 作業内容
1. ActionPanel に「会話をクリア」アクションを追加
2. confirmAlert() で確認ダイアログを表示:
   - タイトル: 「会話をクリアしますか?」
   - メッセージ: 「この操作は取り消せません。」
3. 確認後:
   - state の messages を空配列にリセット
   - clearMessages() で LocalStorage からも削除
   - 成功 Toast を表示

## タスクのチェックリスト
- [ ] ActionPanel に「会話をクリア」アクションが表示される
- [ ] 確認ダイアログが表示される
- [ ] 確認後に会話が全削除される
- [ ] LocalStorage からもデータが削除される
- [ ] クリア後に EmptyView が表示される
