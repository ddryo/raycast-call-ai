# T-M2-2: メインコマンド UI の実装

| 項目 | 内容 |
|------|------|
| マイルストーン | M2 |
| 依存 | T-M2-1 |
| 要件ID | FR-001, FR-002 |

## 説明
List + List.Item.Detail パターンでチャット UI を構築する。SearchBar で質問を入力し、Enter で送信、会話が List に表示され、選択メッセージの内容が右側の Detail に markdown で表示される。

## 作業内容
1. `src/ask-ai.tsx` にメインコマンドコンポーネントを実装
2. List コンポーネントの設定:
   - `isShowingDetail={true}` で右側パネルを有効化
   - `isLoading` を useConversation の状態にバインド
   - `searchBarPlaceholder` に入力ガイドテキストを設定
   - `onSearchTextChange` と `searchText` で入力テキストを state 管理
   - `filtering={false}` で組み込みフィルタリングを無効化
3. SearchBar での送信処理:
   - Enter 押下時（Action 経由）に sendMessage() を呼び出し
   - 送信後に searchText を空にリセット
4. List.Item の構成:
   - messages を逆順（最新が上）で表示
   - title にロール表示（User / AI）
   - subtitle にメッセージのプレビュー（短縮テキスト）
   - accessories に時刻を表示
5. List.Item.Detail の構成:
   - markdown プロパティにメッセージの content をそのまま渡す
