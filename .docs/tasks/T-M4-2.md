# T-M4-2: スレッド一覧表示の実装

| 項目 | 内容 |
|------|------|
| マイルストーン | M4 |
| 依存 | T-M4-1 |
| 要件ID | FR-011 |

## 説明
保存済みのスレッド一覧を表示し、選択してその会話を再開できる UI を実装する。

## 作業内容
1. スレッド一覧の表示方式を実装（ActionPanel → Push で List 表示）:
   - メインの会話画面の ActionPanel に「会話一覧」アクションを追加
   - 選択すると `useNavigation().push()` でスレッド一覧 List へ遷移
   - スレッド選択後はメイン画面に pop で戻り、選択したスレッドの会話が表示される
2. スレッド一覧の List.Item 構成:
   - title: スレッドタイトル
   - subtitle: 最終メッセージのプレビュー
   - accessories: 更新日時
3. スレッド選択時:
   - currentThreadId を切替
   - 該当スレッドの会話を復元して表示

## タスクのチェックリスト
- [ ] スレッド一覧が表示される
- [ ] 各スレッドのタイトルと更新日時が見える
- [ ] スレッドを選択すると、その会話が表示される
- [ ] 現在選択中のスレッドが視覚的に区別できる
