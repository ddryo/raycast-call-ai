# T-M4-3: スレッド削除機能の実装

| 項目 | 内容 |
|------|------|
| マイルストーン | M4 |
| 依存 | T-M4-1 |
| 要件ID | FR-010 |

## 説明
不要な会話スレッドを削除する機能を実装する。

## 作業内容
1. スレッド一覧の ActionPanel に「スレッドを削除」アクションを追加
2. confirmAlert() で確認ダイアログを表示
3. 削除処理:
   - threads から該当スレッドを除去
   - LocalStorage から `ask-ai:messages:{threadId}` を removeItem
   - LocalStorage の threads 一覧を更新
4. 現在表示中のスレッドを削除した場合:
   - 他のスレッドに自動切替（最新のスレッドなど）
   - 切替確定後、LocalStorage の `ask-ai:current-thread` を新しい currentThreadId で更新する
   - スレッドが0件になった場合は新規スレッドを自動作成し、そのスレッドIDを `ask-ai:current-thread` に保存する

## タスクのチェックリスト
- [ ] スレッド一覧から削除アクションを実行できる
- [ ] 確認ダイアログが表示される
- [ ] 削除後に LocalStorage からデータが除去される
- [ ] 現在のスレッド削除時に別スレッドへ自動切替される
- [ ] 全スレッド削除時に新規スレッドが自動作成される
