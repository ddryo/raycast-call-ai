# T-M2-4: 複数行入力フォームの実装

| 項目 | 内容 |
|------|------|
| マイルストーン | M2 |
| 依存 | T-M2-2 |
| 要件ID | FR-005 |

## 説明
SearchBar は単一行入力のため、長文や複数行のテキストを入力したい場合に ActionPanel から Form を開いて送信できるようにする。

## 作業内容
1. ActionPanel に「複数行で入力」アクションを追加
2. Action.Push で Form 画面に遷移:
   - Form.TextArea で複数行テキスト入力
   - 送信ボタンで sendMessage() を呼び出し
   - 送信後に元の List 画面に pop で戻る
3. Form のバリデーション:
   - 空テキストの送信を防止

## タスクのチェックリスト
- [ ] ActionPanel に「複数行で入力」アクションが表示される
- [ ] Form 画面で複数行テキストを入力できる
- [ ] 送信後にメッセージが会話に追加される
- [ ] 送信後に List 画面に戻る
- [ ] 空テキストでは送信できない
