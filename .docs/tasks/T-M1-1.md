## 説明
Raycast 拡張機能のプロジェクトをゼロから作成し、開発に必要な基本設定を整える。
package.json で Preferences（API キー、モデル選択）を定義し、ディレクトリ構成を確立する。

## 作業内容
1. `ray create-extension` または手動で Raycast 拡張機能プロジェクトを作成
2. package.json に拡張機能のメタ情報を記載（name: ask-ai, title, description）
3. package.json の `commands` セクションにメインコマンド `ask-ai` を定義（mode: view）
4. package.json の `preferences` セクションに以下を定義:
   - `apiKey`（type: password, required: true）
   - `model`（type: dropdown, default: gpt-4o-mini）
5. `openai` パッケージを依存に追加
6. src/ 配下のディレクトリ構成を作成（services/, storage/, hooks/, types/）
7. tsconfig.json の設定確認（strict: true）
8. `npm run dev` でビルドが通ることを確認

## タスクのチェックリスト
- [ ] package.json に commands と preferences が正しく定義されている
- [ ] openai パッケージがインストール済み
- [ ] src/ 配下のディレクトリ構成が作成されている
- [ ] `npm run dev` がエラーなく成功する
- [ ] Raycast の設定画面で API Key 入力欄が表示される
